Value 4 Value is a new system for paying podcasters in Bitcoin for their content, using the Lightning Network. The Podcast Index has more info about Value 4 Value and can be used to find supporting podcast players.

This page pulls the current value of BTC in USD from coinbase.com, and converts the desired dollars per hour into SAT per minute (1 BTC = 100,000,000 SAT). Currency.world also has a nice interface for viewing the exchange rate.
